  1. Margin (внешний отступ) - это расстояние между элементом и его соседями, определяющее пространство вокруг элемента. Padding (внутренний отступ) - это расстояние между содержимым элемента и его границей.
  2. Block (блочный), inline (строчный) и inline-block (строчно-блочный) - это способы отображения HTML-элементов. Блочные элементы занимают всю доступную горизонтальную ширину и начинаются с новой строки. Строчные элементы занимают только необходимую ширину и не переносятся на новую строку. Строчно-блочные элементы комбинируют свойства блочных и строчных элементов.
  3. CSS overflow (переполнение) - это свойство CSS, которое определяет, как элемент должен обрабатывать свое содержимое, если оно не помещается внутри его границ. Значения могут быть, например, "visible" (по умолчанию, содержимое переполняет элемент), "hidden" (содержимое скрывается), "scroll" (появляется полоса прокрутки) и др.
  4. CSS float (плавающий элемент) - это свойство CSS, которое позволяет элементам выравниваться по левому или правому краю родительского контейнера и "плавать" внутри него. Это часто используется для создания многоколоночных макетов.
  5. CSS @media rule - это медиа-правило в CSS, которое позволяет применять стили только в зависимости от характеристик устройства или размеров экрана. Например, вы можете использовать @media для создания адаптивных дизайнов, которые будут выглядеть по-разному на мобильных устройствах и на больших экранах.
